# Project SADM

## Current Env
- PHP >= 7.2
- Composer >= 2
- Laravel >= 7.3
- MySQL >= 8
- AdminLTE 3

## Installation
- Clone the repository with `git clone`
- Make a copy of `env.example` and rename it to `.env` and then edit database credentials there.
- run `composer install`
- run `php artisan key:generate`
- run `php artisan migrate`
